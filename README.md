# BiliCache2MP4-GUI
## 前置条件
- Window 10 或更高
- [Python 3.10+](https://mirrors.tuna.tsinghua.edu.cn/python/3.14.4/python-3.14.4-amd64.exe)
- [ffmpeg 8.0+](https://github.com/GyanD/codexffmpeg/releases/download/8.1/ffmpeg-8.1-essentials_build.7z) （如果下载速度慢，尝试使用[ghproxy](https://ghproxy.link/)之类的镜像网站）

## 下载本体

<https://github.com/kaixinol/BiliCache2MP4-GUI/releases/>
